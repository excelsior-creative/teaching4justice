# Teaching4Justice CEO Folder

This folder is the repo-local operating layer for `teaching4justice` under the Excelsior Business Operating System.

Start here:

1. `config.md` — authoritative repo context and default risk tier.
2. `conventions.md` — repo-specific operating rules.
3. `next.md` — executable queue.
4. `specs/` — scoped implementation specs.
5. `INBOX.md` — questions, blockers, and classification notes for Brandon.

The base shape follows BOS `01-foundations/04-ceo-folder-pattern.md`.
